Documentation

chainchainchain gives you live, dynamic and non-linear pseudo-inheritance.

Consider the objects A, B and C; this means that B can inherit from A and C can inherit from B without having to inherit from A. You can define the lineage of any object as you like, not interfering with any other objects lineage. You can have all sorts of forks, loops and paradoxes in the inheritance tree. If you change object A's properties, B's link to it remains live. You can even change the whole lineage of an object reestructuring the whole inheritance chain whenever you like. This is the power of chainchainchain.

.chain ( obj , obj , ... ) Adds objects to the top of the inheritance chain.
obj object
Objects from wich to inherit.
returns this
Mind that the rightmost objects always become the oldest in chain as this method adds to the top of the chain. If you want to add to the bottom of the chain, use .inchain().
.dechain ( obj , obj , ... ) Removes objects from the inheritance chain.
obj object
Objects to remove. Clears the chain when called with no arguments.
returns this
.rechain ( x , o ) Replaces object in chain.
x object
Object to replace from chain.
o object
Object to replace x with.
returns this
.inchain ( obj , obj , ... ) Adds objects to the bottom of the inheritance chain.
obj object
Objects from wich to inherit.
returns this
Mind that the leftmost objects always become the youngest in chain after the caller, as this method adds to the bottom of the chain. If you want to add to the top of the chain, use .chain().
.get ( prop , own ) Gets the youngest selected property in the chain.
prop string
Name of the property to return.
own boolean
If true , methods are applied to the object calling .get(), even if the owner is up the chain.
returns value
Mind that this method looks for properties only up the caller's chain. To look up the whole chain tree use .getget().
.getget ( prop , own ) Gets the youngest selected property in the chain tree.
prop string
Name of the property to return.
own boolean
If true , methods are applied to the object calling .getget(), even if the owner is up the chain tree.
returns value
Mind that the younger chained objects's whole chains are considered younger than older chained objects.
Use with caution! This method is recursive. Watch out for loops in the chain tree. Use with chain.allowloops = false.
.raw ( prop , own ) Gets each chained object's selected property.
prop string
Name of the property or method you want to return.
own boolean
If true , methods are applied to the object calling .raw(), even if the owner is up the chain.
returns array
Mind that this method looks for properties only up the caller's chain. To look up the whole chain tree use .rawraw().
.raw(prop,own)[0] and .get(prop,own) return the same.
.rawraw ( prop , own ) Getgets each chained object's selected property.
prop string
Name of the property or method you want to return.
own boolean
If true , methods are applied to the object calling .rawraw(), even if the owner is up the chain tree.
returns array
Use with caution! This method uses .getget() wich is recursive. Use with chain.allowloops = false.
.rawraw(prop,own)[0] and .getget(prop,own) return the same.
